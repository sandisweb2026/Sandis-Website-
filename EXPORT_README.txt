Sandis Tours Backend + Admin + Database Export

Included:
- server/ backend Express API code
- api/ API deployment files
- src/ full frontend/admin React source code
- public/ public assets
- database/sandis-database-dump.sql current MySQL schema + data dump
- database/sandis-original.sql original repository SQL file
- package.json, package-lock.json, config files, setup docs

Excluded intentionally:
- node_modules/ because dependencies reinstall with npm install
- dist/ because it is generated by npm run build
- .git/ repository history
- .env because it can contain local secrets; use .env.example
- log files, screenshots, and previous zip files

Run after extracting:
1. npm install
2. create/import MySQL database from database/sandis-database-dump.sql
3. copy .env.example to .env and update DB values
4. npm run dev:full
